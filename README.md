# rfr16e

16 key macropad QMK-based

This macropad uses STM32F401CCU6 "WeAct BlackPill" and QMK Firmware. DIP Switch can be used to change layer/keymap.
Simple 16 key macropad with alternate layout 16 key + 1 encoder.
